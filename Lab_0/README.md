# Лабораторная работа 1
## В данной лабороторной работе мы создаем 2 простых React компонента
### Компонент Greeting, который отображает приветствие в зависимости от времени суток
### Компонент ProfileCard, который отображает карточку профиля пользователя с его именем, аватаркой и краткой информацией о нем
### Для стилизации компонентов используется CSS-in-JS подход с использованием встроенных стилей в самих компонентах.
